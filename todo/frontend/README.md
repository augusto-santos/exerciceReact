### Iniciando
  Instale as dependencias do Node:
```
npm install
```
  Iniciando aplicação no **modo de desenvolvedor**
```
npm run dev
```
  A aplicação estará funcionando no [http://localhost:8080](http://localhost:8080)

  **Build de produção**
```
npm run production
```
  Os arquivos de produção estarão na pasta __public__
